# 首页模式说明文案设计

**Goal:** 移除任何会让用户误以为儿童模式会自动替玩家操作的介绍，并在首页与 README 中补齐三档模式的清晰说明。

**Context:**
- 当前三档难度的实际差异主要来自机器人选牌策略，以及儿童模式额外展示的引导文案。
- `src/game/assists/child-mode.ts` 中保留了自动借牌、自动出牌的钩子，但当前实现均返回空值，不应再在用户文案中暗示这些能力存在。
- 首页目前只有难度选择按钮，没有补充解释三档差异；README 也只提到“支持三档难度”，缺少模式说明。

**Design:**
- 首页在难度选择区域下方新增一块静态模式说明，分别解释 `儿童 / 标准 / 挑战` 的定位。
- 文案明确三档区别是“引导强度”和“机器人策略强度”，不再提及任何自动代打、自动借牌。
- README 增加与首页一致的模式说明，保证仓库介绍和实际体验口径一致。

**Non-Goals:**
- 不修改难度算法或实际游戏规则。
- 不删除 `child-mode.ts` 中现有钩子代码，本次只调整用户可见文案。

**Validation:**
- 更新首页流程测试，断言模式说明可见。
- 运行相关 Vitest 测试，确认首页仍可正常开始对局。
