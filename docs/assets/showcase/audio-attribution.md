# Audio Attribution

The showcase sound effects in `public/assets/audio/` are original assets generated locally for this project with `ffmpeg`'s built-in synthesis filters.

- `ui-click.mp3` - generated in-repo, original project asset
- `game-start.mp3` - generated in-repo, original project asset
- `card-play.mp3` - generated in-repo, original project asset
- `card-borrow.mp3` - generated in-repo, original project asset
- `round-end.mp3` - generated in-repo, original project asset

No third-party source recordings or sample packs are embedded in these files.
